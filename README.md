# 🐾 Paw & Clover – Pet Store Website

Paw & Clover is a multi-page pet store website designed to simulate a real-world e-commerce experience.  
The project focuses on clean UI design, structured navigation, and interactive features like product filtering and search.

## ✨ Features

- Multi-page layout (Cats, Dogs, categories)
- Product filtering system (type + price)
- Live search functionality across products
- Organized category navigation (food, toys, litter, beds)
- Responsive, clean UI design
- Reusable components and consistent styling

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 Purpose

This project was built to practice:
- Structuring larger multi-page websites
- Creating interactive UI features with JavaScript
- Designing realistic small business / e-commerce layouts

- Add shopping cart functionality
- Add product detail pages
- Connect to real product API or database
